# Stack(스택)
## 개념
- 후입선출(Last-In First-Out)
- 스택: 쌓아놓은 더미
- 자료의 삽입과 삭제를 한쪽으로만 하는 선형구조
- 가장 나중에 삽입된 자료가 가장 먼저 삭제됨(항아리)

## Stack ADT(규칙 나열)
- Stack(): 비어 있는 새로운 스택 리스트로 구현
- isEmpty(): 비어 있으면 True, 아니면 False 리턴
- push(e): 항목 e를 스택 맨 위에 추가
- pop(): 스택의 맨 위에 있는 항목을 꺼내 리턴
- peek(): 스택의 맨 위에 있는 항목을 삭제하지 않고 알려줌
- size(): 스택 내의 모든 항목의 개수 리턴
- clear(): 스택을 공백 상태로 만듦

## 용도
- 자료의 출력 순서가 입력의 역순이어야 할 때
- 편집기의 되돌리기 기능
- 함수호추에서 복귀주소 저장
- 괄호 검사
- 계산기
- 미로 탐색 등
